# Day 1: Python 第一课应该掌握什么

如果你今天没怎么学，那第一天其实不用追求太多。

先把下面这 6 件事学明白，就已经够好了：

## 1. 变量

- 变量就是给数据起名字
- 例如：`name = "Minda"`
- 左边是变量名，右边是值

你今天至少要知道：

- 字符串 `str`
- 整数 `int`
- 浮点数 `float`
- 布尔值 `bool`
- 空值 `None`

## 2. 函数

- 函数就是把一段逻辑封装起来
- 定义函数用 `def`
- 调用函数就是写函数名加括号

例如：

```python
def greet(name: str) -> str:
    return f"你好，{name}"
```

你今天至少要理解：

- 什么是参数
- 什么是返回值
- 为什么要把重复逻辑写成函数

## 3. 模块

- 一个 `.py` 文件就是一个模块
- 可以在一个文件里写函数
- 然后在另一个文件里用 `import` 或 `from ... import ...` 导入

这也是为什么当前目录里有：

- `hello.py`：主示例文件
- `test.py`：被导入的模块示例

## 4. 容器类型

Day 1 最重要的是下面 4 个：

### 列表 `list`

- 有顺序
- 可以重复
- 可以修改
- 适合保存“一组任务”

例如：

```python
tasks = ["安装 Python", "创建虚拟环境", "运行 hello.py"]
```

### 字典 `dict`

- 键值对结构
- 适合描述“一个对象”

例如：

```python
student = {"name": "Minda", "goal": "学习 Python 后端"}
```

### 元组 `tuple`

- 有顺序
- 不可修改
- 适合保存固定数据

例如：

```python
screen_size = (1920, 1080)
```

### 集合 `set`

- 无序
- 不重复
- 常用来去重

例如：

```python
tags = {"python", "python", "fastapi"}
```

## 5. 条件和循环

### 条件判断

- `if`
- `elif`
- `else`

用来根据不同情况执行不同逻辑。

### 循环

- `for`
- `while`

Day 1 先重点掌握 `for`。

例如：

```python
for task in tasks:
    print(task)
```

## 6. 入口写法

你今天一定要见一次这句：

```python
if __name__ == "__main__":
    main()
```

它的作用是：

- 直接运行当前文件时，执行 `main()`
- 如果这个文件只是被别人导入，就不自动执行

这是 Python 项目里非常常见的写法。
