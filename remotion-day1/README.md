# Remotion Day 1 Video

这个子项目把 `2026-04-16-day1` 的文档和示例代码整理成一个可渲染的 Remotion 教学视频。

当前版本已经包含：

- 分章节时间轴
- 字幕节奏条
- 中文配音稿
- 更丰富的动态背景、章节进度条、代码高亮和轨道动画
- 一个额外的创意长视频 `CosmicDogOdyssey`

## 命令

```bash
npm install
npm run compositions
npm run preview
npm run render
```

## 输出

- Composition ID: `Day1PythonLesson`
- 默认输出: `out/day1-python-lesson.mp4`
- 中文配音稿: `NARRATION_ZH.md`
- 新创意视频 Composition: `CosmicDogOdyssey`
