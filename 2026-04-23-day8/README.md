# Day 8: MySQL 基础入门

Day 8 开始进入数据库阶段。

今天重点不是上 ORM，而是先把 MySQL 最基础的东西打牢。

## 你今天要弄明白的概念

- 数据库是什么
- 表是什么
- 字段是什么
- 主键是什么
- 唯一键是什么
- 最基本的 SQL 增删改查怎么写

## 建议学习顺序

1. 先理解数据库、表、字段关系
2. 再看 `users` 表怎么设计
3. 再练 `INSERT / SELECT / UPDATE / DELETE`
